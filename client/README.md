Just-divide

This is a  ReactJS-based puzzle game where players use division logic to merge tiles and clear the grid.

Some of its features are:

Drag & drop tile placement
 Division-based merge logic
 Queue system (upcoming tiles)
 KEEP slot (store & swap tiles)
 Score & level progression
 Game over detection
Responsive design

Its technicality is as follows:

 Equal tiles disappear
 Divisible tiles merge (e.g. 12 ÷ 3 = 4)
 Result of 1 is removed
 Only the top tile from the queue is playable

To run the game, use the following commands:


npm install
npm run dev
